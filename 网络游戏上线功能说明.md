# 网络游戏上线功能说明

## 功能概述

针对网络游戏的特殊商业模式，实现了差异化的游戏上线流程和付费内容推荐系统。

## 主要功能

### 1. 数据概览显示差异化

**网络游戏显示：**
- **数据概览**（代替"收益概览"）
- **总注册**（显示总注册用户数，代替"总收益"）
- **当前在线**（显示当前在线人数，约为总注册数的10%，代替"总销量"）

**单机游戏保持不变：**
- 收益概览
- 总收益
- 总销量

### 2. 游戏上线对话框差异化

#### 单机游戏（保持原有逻辑）
- 标题：**🎮 游戏发售**
- 显示价格推荐和价格区间
- 需要设置发售价格
- 按钮：**🚀 确认发售 (¥XX)**

#### 网络游戏（新功能）
- 标题：**🎮 游戏上线**
- **不显示发售价格输入区域**
- 显示付费内容建议卡片，包含：
  - **道具价格梯度：**
    - 小额道具（如小额消费道具）
    - 中档道具（如礼包、皮肤）
    - 高档道具（如稀有道具）
  - **VIP订阅价格：**
    - 月卡
    - 季卡
    - 年卡
  - **市场分析**（根据游戏平台和主题提供个性化建议）
- 按钮：**🚀 立即上线**（无需输入价格，直接可以点击）

### 3. 付费内容智能推荐算法

基于以下因素计算推荐价格：
- **平台系数**：
  - PC：1.0x（基准）
  - 手机：0.6x
  - 主机：1.5x
  - 网页：0.4x
- **主题受欢迎度**：
  - 动作、RPG等受欢迎主题有更高系数
  - 休闲、解谜等主题系数较低

#### 价格梯度
- **小额道具**：基础价格 × 0.5
- **中档道具**：基础价格 × 1.0
- **高档道具**：基础价格 × 3.0
- **月卡VIP**：基础价格 × 4.0
- **季卡VIP**：月卡 × 2.5
- **年卡VIP**：月卡 × 8.0

#### 市场分析示例

**动作类PC网游：**
> "PC平台付费能力适中，平衡定价策略。动作游戏适合皮肤、特效等视觉类付费内容。建议同时提供道具和VIP订阅，满足不同玩家需求。"

**策略类手机网游：**
> "移动/网页平台用户量大，建议小额高频付费。策略游戏适合加速、资源等便利类付费内容。建议同时提供道具和VIP订阅，满足不同玩家需求。"

## 技术实现

### 新增数据类

**MonetizationRecommendation**（付费内容推荐）
```kotlin
data class MonetizationRecommendation(
    val gameId: String,
    val itemPrices: ItemPriceRecommendation,
    val vipPrices: VipPriceRecommendation,
    val marketAnalysis: String,
    val confidence: Float = 0.8f
)
```

**ItemPriceRecommendation**（道具价格推荐）
```kotlin
data class ItemPriceRecommendation(
    val lowTier: Float,  // 低档道具价格
    val midTier: Float,  // 中档道具价格
    val highTier: Float  // 高档道具价格
)
```

**VipPriceRecommendation**（VIP价格推荐）
```kotlin
data class VipPriceRecommendation(
    val monthly: Float,   // 月卡价格
    val seasonal: Float,  // 季卡价格
    val yearly: Float     // 年卡价格
)
```

### 核心方法

**PriceRecommendationEngine.calculateMonetizationRecommendation()**
- 计算网络游戏的付费内容推荐价格
- 返回道具和VIP价格建议
- 生成个性化市场分析

### 修改的文件

1. **GameData.kt** - 添加新的数据类定义
2. **PriceRecommendationEngine.kt** - 添加付费内容推荐算法
3. **GameReleaseDialog.kt** - 实现网络游戏和单机游戏的差异化UI
4. **EnhancedGameProjectCard.kt** - 实现数据概览的差异化显示

## 使用流程

1. 开发完成网络游戏（进度100%）
2. 系统自动弹出"游戏上线"对话框
3. 查看付费内容建议（道具价格和VIP订阅价格）
4. 参考市场分析了解定价策略
5. 点击"立即上线"按钮（无需输入价格）
6. 游戏正式上线，开始运营

## 设计理念

- **符合真实商业模式**：网络游戏通常免费下载，通过内购和订阅获利
- **智能推荐**：根据游戏类型和平台提供个性化建议
- **简化流程**：网络游戏上线无需设置买断价格，降低操作复杂度
- **数据可视化**：清晰展示不同档次的付费内容价格，便于决策
