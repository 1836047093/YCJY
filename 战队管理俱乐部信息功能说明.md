# 战队管理俱乐部信息功能说明

## 功能概述

在战队管理界面新增了"俱乐部信息"页签，玩家可以查看当前游戏中所有的电竞战队及其所属公司信息。

## 主要功能

### 1. 俱乐部信息Tab

新增第5个Tab页签"俱乐部信息"，位于"英雄图鉴"之后。

### 2. 战队列表展示

- **玩家战队**：特殊高亮显示，使用绿色边框和深蓝色背景
  - 显示"我的战队"标签
  - 所属公司显示为"玩家公司"
  - 实时显示当前战队人数和平均实力

- **AI战队**：显示15支AI战队
  - 随机分配给不同的游戏公司（腾讯、网易、米哈游等）
  - 每支战队有独特的名称（如"龙之战队"、"凤凰俱乐部"等）
  - 显示战队实力评分和冠军数

### 3. 搜索和筛选功能

- **搜索框**：支持按战队名称或公司名称搜索
- **排序选项**：
  - 按战队名排序
  - 按公司名排序
  - 按实力排序（降序）

### 4. 战队信息卡片

每个战队卡片显示：
- 战队名称和图标
- 所属公司
- 队员数量
- 平均实力评分
- 冠军数量（如有）

### 5. 视觉设计

- 玩家战队使用绿色主题，突出显示
- AI战队使用深色主题
- 冠军战队显示金色奖杯标识
- 实力评分使用颜色区分：
  - 80+：橙色（强队）
  - 70-79：绿色（中等）
  - 70以下：灰色（弱队）

## 技术实现

### 数据结构

```kotlin
data class EsportsTeamInfo(
    val teamName: String,           // 战队名称
    val companyName: String,        // 所属公司
    val isPlayerTeam: Boolean,      // 是否玩家战队
    val playerCount: Int,           // 队员数量
    val averageRating: Int,         // 平均实力
    val championships: Int          // 冠军数
)
```

### 主要组件

1. **ClubInfoTab**：俱乐部信息主界面
2. **TeamInfoCard**：战队信息卡片组件
3. **TeamStatColumn**：战队统计数据列组件

## 使用方式

1. 进入"战队管理"界面
2. 点击顶部的"俱乐部信息"Tab
3. 浏览所有战队信息
4. 使用搜索框查找特定战队或公司
5. 使用排序按钮调整显示顺序
