# 战队解锁存档修复说明

## 问题描述
用户解锁并创建了战队，但保存后再次加载存档，战队解锁状态被重置。

## 根本原因
在保存存档时，`esportsTeamUnlocked` 字段没有被包含在 `SaveData` 构造函数中，导致战队解锁状态无法保存到存档文件。

## 修复内容
在所有保存存档的地方添加了 `esportsTeamUnlocked` 字段：

### 1. 自动存档（第3165行）
```kotlin
esportsTeamUnlocked = esportsTeamUnlocked,
```

### 2. 手动存档（覆盖模式）（第9524行）
```kotlin
esportsTeamUnlocked = esportsTeamUnlocked, // 保存战队管理解锁状态
```

### 3. 手动存档（新存档）（第9670行）
```kotlin
esportsTeamUnlocked = esportsTeamUnlocked, // 保存战队管理解锁状态
```

## 修复效果
- ✅ 战队解锁状态现在可以正确保存到存档
- ✅ 重新加载存档后战队解锁状态不会丢失
- ✅ 已解锁的战队功能在重新加载后依然可用
- ✅ 完全向后兼容，不影响现有存档

4. 验证战队管理界面是否依然可以访问
