# GM工具箱 - 一键满配战队（快速说明）

## ✅ 已完成

在GM工具箱中新增了**一键满配战队成员**功能。

## 🎯 功能说明

点击按钮后自动：
1. 清空现有战队成员
2. 为每个位置（上单、打野、中单、ADC、辅助）招募1名SSR选手
3. 自动签约所有新选手
4. 总计创建5名SSR品质的顶级选手（每个位置1名，不重复）

## 🚀 使用方法

1. 设置 → 兑换码 → 输入 `progm` → 激活GM模式
2. 设置 → GM工具箱 → 点击 **⚽ 一键满配战队成员**
3. 前往 赛事 → 战队管理 → 查看新战队

## 📋 修改文件

- `MainActivity.kt` - 添加GM按钮和回调逻辑
- `PlayerManager.kt` - 扩展招募方法支持指定位置

## ⚠️ 注意

- 会清空现有战队成员（但不删除，仍在全部选手列表）
- 创建的选手会保存到存档
- 仅用于测试，会破坏游戏平衡性

## 🎨 界面

```
🛠️ GM工具箱

[📊 修改游戏数据]
[💰 一键加1000万]
[⭐ 各职位6名5级员工]
[⚽ 一键满配战队成员] ← 新增（蓝色按钮）
```
