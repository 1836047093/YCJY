# 竞争对手图标设计说明

## 概述
为9个竞争对手公司设计了独特且富有创意的专属图标，每个图标都基于公司名称的含义和特点精心设计。

## 图标设计方案

### 1. 光源互娱 💡
- **图标**: 灯泡
- **寓意**: 象征光源与创意的诞生，代表创新和灵感
- **特点**: 明亮、创新、引领潮流

### 2. 棱镜游戏 🔷
- **图标**: 蓝色菱形
- **寓意**: 象征棱镜的多面折射效果，代表多元化视觉体验
- **特点**: 多彩、视觉化、艺术性

### 3. 梦境互娱 🌙
- **图标**: 月亮
- **寓意**: 象征梦境与幻想世界，代表神秘和想象力
- **特点**: 梦幻、神秘、富有想象力

### 4. 星尘枢纽 ⭐
- **图标**: 星星
- **寓意**: 象征星尘与宇宙枢纽，代表连接和汇聚
- **特点**: 闪耀、汇聚、宏大

### 5. 微光火花 ✨
- **图标**: 闪光
- **寓意**: 象征微小但耀眼的灵感火花，代表创意的迸发
- **特点**: 灵动、创意、充满活力

### 6. 泰坦互娱 ⚡
- **图标**: 闪电
- **寓意**: 象征泰坦的强大力量，代表爆发力和冲击力
- **特点**: 强大、迅猛、震撼

### 7. 乱线工坊 🧵
- **图标**: 线
- **寓意**: 直接对应"乱线"，象征工坊的精细编织与创作
- **特点**: 精细、手工、独立创作

### 8. 璀璨游戏 💎
- **图标**: 钻石
- **寓意**: 象征璀璨夺目的高品质，代表珍贵和卓越
- **特点**: 高端、精致、闪耀

### 9. 神盾动力 🛡️
- **图标**: 盾牌
- **寓意**: 直接对应"神盾"，象征防护力量和稳定性
- **特点**: 稳固、可靠、保护

## 技术实现

### 修改的文件
- `CompetitorData.kt`

### 主要改动
1. **函数重构**: 将 `getRandomCompanyLogo()` 改为 `getCompanyLogo(companyName: String)`
2. **固定映射**: 通过 `when` 表达式建立公司名称到图标的固定映射关系
3. **调用更新**: 在 `generateInitialCompetitors()` 中传入公司名称参数

### 代码示例
```kotlin
private fun getCompanyLogo(companyName: String): String {
    return when (companyName) {
        "光源互娱" -> "💡"  // 灯泡，象征光源与创意
        "棱镜游戏" -> "🔷"  // 菱形，象征棱镜的多面折射与视觉效果
        "梦境互娱" -> "🌙"  // 月亮，象征梦境与幻想
        "星尘枢纽" -> "⭐"  // 星星，象征星尘与宇宙枢纽
        "微光火花" -> "✨"  // 闪光，象征微光与灵感火花
        "泰坦互娱" -> "⚡"  // 闪电，象征泰坦的强大力量
        "乱线工坊" -> "🧵"  // 线，象征工坊的精细编织与创作
        "璀璨游戏" -> "💎"  // 钻石，象征璀璨夺目的品质
        "神盾动力" -> "🛡️" // 盾牌，象征神盾的防护与力量
        else -> "🎮"        // 默认游戏图标
    }
}
```

## 设计原则

1. **名称关联性**: 每个图标都与公司名称有直接或间接的关联
2. **独特性**: 9个图标各不相同，易于识别和区分
3. **视觉冲击**: 选择色彩鲜明、特征明显的emoji
4. **文化契合**: 符合游戏公司的行业特点和文化属性
5. **固定性**: 确保同一公司在所有游戏中始终使用相同图标

## 效果
- ✅ 每个竞争对手都有专属的、固定的图标
- ✅ 图标设计富有创意，与公司名称高度契合
- ✅ 提升了竞争对手的识别度和品牌感
- ✅ 在排行榜、动态新闻等界面中更易于识别
