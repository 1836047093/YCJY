## 二、成功等级倍率

| 成功等级 | 收益倍率 | 效果倍率（粉丝/玩家/兴趣值） |
|---------|---------|---------------------------|
| 大成功 | 1.3倍 | 1.5倍 |
| 成功 | 1.0倍 | 1.0倍 |
| 一般 | 0.8倍 | 0.7倍 |
| 失败 | 0.5倍 | 0.5倍 |

## 三、收益计算公式

### 1. 赞助商收入
```
赞助商收入 = 投资金额 × 基础赞助率 × 评分倍率 × 玩家倍率 × 成功等级倍率

其中：
- 基础赞助率 = (sponsorRateMin + sponsorRateMax) / 2.0
- 评分倍率 = 游戏评分 / 10.0 (默认6.0分)
- 玩家倍率 = 1.0 + log10(活跃玩家数 / 10000) × 0.1
- 成功等级倍率 = revenueMultiplier
```

### 2. 转播权收入
```
转播权收入 = 基础转播权收入 × (1.0 + 活跃玩家数 / 1000000 × 0.2) × 成功等级倍率
```

### 3. 门票收入（仅国际赛和全球赛）
```
普通票数 = 活跃玩家数 × 0.0001
VIP票数 = 活跃玩家数 × 0.00001
普通票价 = 200-500 (随机)
VIP票价 = 1000-3000 (随机)

门票收入 = (普通票数 × 普通票价 + VIP票数 × VIP票价) × 成功等级倍率
```

### 4. 总收益
```
总收益 = 赞助商收入 + 转播权收入 + 门票收入
净利润 = 总收益 - 投资金额
```

## 四、粉丝增长计算公式

```
粉丝增长 = 当前粉丝数 × 粉丝增长比例 × 成功等级效果倍率 × 随机因子

其中：
- 粉丝增长比例 = (fansGrowthMin + fansGrowthMax) / 2.0
- 成功等级效果倍率 = effectMultiplier
- 随机因子 = 1.0 + Random(-0.1, 0.1) = 0.9 ~ 1.1
```

## 五、典型场景收益示例

### 场景1：城市杯（成功等级：成功）

**假设条件：**
- 游戏评分：8.0分
- 活跃玩家：2万
- 当前粉丝：10万
- 投资金额：¥50万

**计算过程：**

1. **赞助商收入**
   - 基础赞助率 = (0.30 + 0.50) / 2 = 0.40
   - 评分倍率 = 8.0 / 10.0 = 0.8
   - 玩家倍率 = 1.0 + log10(20000 / 10000) × 0.1 = 1.0 + log10(2) × 0.1 ≈ 1.03
   - 成功等级倍率 = 1.0
   - 赞助商收入 = 500000 × 0.40 × 0.8 × 1.03 × 1.0 ≈ **¥16.5万**

2. **转播权收入**
   - 基础转播权收入 = 0
   - 转播权收入 = **¥0**

3. **门票收入**
   - 门票收入 = **¥0**（城市杯无门票）

4. **总收益**
   - 总收益 = 16.5万 + 0 + 0 = **¥16.5万**
   - 净利润 = 16.5万 - 50万 = **-¥33.5万**（亏损）

5. **粉丝增长**
   - 粉丝增长比例 = (0.05 + 0.10) / 2 = 0.075
   - 成功等级效果倍率 = 1.0
   - 随机因子 ≈ 1.0（取平均值）
   - 粉丝增长 = 100000 × 0.075 × 1.0 × 1.0 = **+7,500粉丝**

## 七、收益影响因素总结

### 1. 收益影响因素（按重要性排序）
1. **成功等级** - 影响所有收益的倍率（0.5倍 - 1.3倍）
2. **游戏评分** - 影响赞助商收入（评分越高，赞助商收入越多）
3. **活跃玩家数** - 影响所有三类收入（玩家越多，收益越高）
4. **投资金额** - 影响赞助商收入（投资越多，赞助商收入越多）

### 2. 粉丝增长影响因素
1. **成功等级** - 影响效果倍率（0.5倍 - 1.5倍）
2. **当前粉丝数** - 粉丝越多，增长绝对值越大
3. **赛事类型** - 更高级的赛事增长比例更高

### 3. 成功等级判定因素
- 基础分：50分
- 游戏评分影响：0-20分（评分越高越好）
- 公司声誉影响：0-15分（声誉越高越好）
- 投入成本奖励：2-10分（更高级的赛事奖励更多）
- 随机因素：0-5分

## 八、注意事项

1. **门票收入**：只有国际赛和全球总决赛才有门票收入
2. **转播权收入**：城市杯没有转播权收入
3. **随机因子**：粉丝和玩家增长有±10%的随机波动
4. **门票价格**：门票价格是随机的，普通票200-500，VIP票1000-3000
5. **亏损风险**：低级赛事（城市杯、全国锦标赛）可能亏损，需要评分和玩家数足够高才能盈利
6. **冷却时间**：每种赛事都有冷却时间，不能连续举办

## 九、收益优化建议

1. **优先提升游戏评分**：评分越高，赞助商收入越多
2. **积累活跃玩家**：玩家数越多，所有收入都越高
3. **提升公司声誉**：提高成功等级概率，获得更高收益倍率
4. **选择合适的赛事**：根据玩家数和粉丝数选择最合适的赛事类型
5. **避免过早举办高级赛事**：如果玩家数不足，高级赛事可能亏损严重
